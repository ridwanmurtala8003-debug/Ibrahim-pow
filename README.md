# Ibrahim Murtala Web3 POW Website

This is a deploy-ready Vite + React + Tailwind portfolio website.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Deploy

Upload this folder to GitHub and deploy on Vercel or Netlify.
